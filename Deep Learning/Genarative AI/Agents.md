- We can think of agents as enabling “tools” for LLMs. Like how a human would use a calculator for math's or perform a Google search for information — agents allow an LLM to do the same thing.
#### Tool :
- a function or a method an agent can use when it is necessary
# Agent Types:

1.  **Zero-Shot Agent**
	
	- An agent that can **perform tasks without prior training** on them.
	- Uses **generalization from related knowledge** or pre-trained models (e.g., GPT-4, CLIP).
	- Example: A model trained on English that can translate unseen languages without fine-tuning.

2. **Few-Shot Agent**
	
	- Learns a new task with **very few examples** (unlike traditional ML, which needs lots of data).
	- Example: A chatbot learning slang from just a few user interactions.

3. **Self-Supervised Agent**

	- Learns from **unlabeled data** by generating its own training signals.
	- Example: BERT (Bidirectional Encoder Representations from Transformers) predicting missing words in sentences.

4. **Multi-Modal Agent**

	- Can process and understand multiple data types like **text, images, audio, and video**.
	- Example: OpenAI’s CLIP, which links images to text descriptions.

5. **Autonomous Learning Agent**

	- Continuously **learns and adapts** without human supervision.
	- Example: Reinforcement Learning (RL) agents in robotics or self-driving cars.

6. **Foundation Model Agent**
	
	- Built on massive **pre-trained models** that can handle multiple downstream tasks.
	- Example: GPT-4, Llama 3, and Gemini 1.5 acting as general-purpose AI assistants.
